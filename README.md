# MoviesExplorerFrontend
Дипломный проект Практикума, который включает в себя:

▪ адаптивную верстку;

▪ регистрацию;

▪ авторизацию;

▪ создание карточки с фильмом;

▪ удаление карточки фильма с профиля;

▪ поиск фильмов;
